# Evolivie Santé

Application React + TypeScript + Vite + Tailwind CSS pour le tunnel de souscription Néoliane.

## Développement

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Configuration

Copiez `.env.example` vers `.env` et renseignez votre `VITE_NEOLIANE_TOKEN`.
